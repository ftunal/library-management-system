# 📚 Library Management System

A desktop Library Management System developed in Java using JavaFX.

This project was developed for the **Design and Analysis of Information Systems** course at the **Polish-Japanese Academy of Information Technology (PJATK), Warsaw**.

---

# ✨ Features

- 📖 Book Management
- 👤 Customer Management
- 🛒 Shopping Basket
- 💳 Payment System
- 🚚 Delivery Management
- 🔔 Notification System
- 🖥️ JavaFX Graphical User Interface
- 💾 Data Serialization
